# 🦒 兒童成長小幫手 (Taiwan Child Growth Helper) v3.0

一款專為台灣家長與醫療專業人士設計的智慧型兒童生長評估工具。結合台灣本土研究數據與臨床骨齡預估模型，守護孩子健康成長。

---

## 🌟 核心特色與功能

### 1. 智慧生長曲線分析
*   **即時百分位計算**：輸入身高、體重與年齡，立即對照衛福部標準計算百分位。
*   **BMI 健康評估**：自動計算 BMI 並判定體位狀態（過輕、正常、過重、肥胖）。
*   **動態視覺化圖表**：使用 D3.js 繪製生長曲線圖，直觀掌握孩子在同年齡層中的位置。
*   **早產兒矯正年齡：支援 3 歲以下 早產兒矯正年齡計算，提供更準確的生長判讀。

### 2. 雙軌成年身高預測
*   **遺傳目標身高 (TH)**：根據父母身高，使用 Tanner 遺傳公式預估孩子的遺傳潛力範圍。
*   **臨床骨齡預估 (PAH)**：整合 Bayley-Pinneau 模型，結合骨骼成熟度（骨齡）推估最終身高。

### 3. 深度發育偏差預警 (v3.0 亮點)
*   **骨齡 vs 遺傳對照**：自動比對「預估身高」與「遺傳範圍」，判定發育動能是否符合遺傳基準。
*   **五級風險標籤**：自動標記「超標預警、優於遺傳、符合預期、略低於潛力、顯著落後」。
*   **發育節奏提醒**：即時提示「發育偏快（留意提早成熟）」或「發育偏慢（動力較弱）」，方便後續追蹤。

### 4. 專業衛教與紀錄管理
*   **個人化建議系統**：針對不同生長狀況提供中西醫結合的調養指南。
*   **一鍵資料匯出**：支援 CSV 格式匯出，方便家長帶至診間與醫師進行深度討論。
*   **常見問題 Q&A**：完整的計算邏輯說明與發育知識庫。

## 📊 專業資料來源與判定標準

本工具之判定標準遵循 **衛生福利部國民健康署** 公告之建議值與臨床通用模型：

* **0–5 歲生長標準**：採用世界衛生組織 (WHO) 公布之「國際嬰幼兒生長標準」。
* **7–18 歲生長標準**：引用自 **陳偉德醫師** 與 **張美惠醫師** 於 2010 年發表之台灣兒童與青少年成長曲線研究。
* **5–7 歲銜接邏輯**：參考 WHO BMI Rebound 趨勢進行數據平滑銜接。
* **身高預測模型**：基於 Tanner 遺傳目標身高公式 (TH) 及 **Bayley–Pinneau (BP)** 臨床預測模型。
* **早產兒校正**：針對 **3 歲以下** 早產兒提供矯正年齡換算，以 40 週預產期為基準點。
